# GPT-2 in zig

The goal of this experiment/project is to implement GPT-2 from scratch in zig and in that process, learn the things involved in training a modern day LLM.

I will mark comments within the code itself as my notes of learning. If there isn't a comment in a place, then it might be a very obvious statement that I know.

I am following the [GOAT](https://www.youtube.com/watch?v=l8pRSuU81PU) and using Gemini+Google Search for exploring zig related info and the exact implementation details of Python methods used in the code (from pytorch, etc.).
